# Calculadora Simples com React Native

Este é um projeto de exercício da aula de desenvolvimento mobile, implementando uma calculadora simples utilizando React Native com Expo e React Native Paper.

## Objetivo

Compreender o uso de componentes, frameworks visuais e Hooks em React Native através da criação de uma calculadora funcional com operações básicas.

## Funcionalidades

- Interface visual com botões numéricos e operadores
- Display para exibição dos valores e resultados
- Operações básicas: adição (+), subtração (-), multiplicação (*), divisão (/)

## Tecnologias Utilizadas

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [React Native Paper](https://reactnativepaper.com/)

## Pré-requisitos

- Node.js instalado
- Expo CLI instalado globalmente (`npm install -g @expo/cli`)
- Git

## Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/Gutinjao22/Calculadora-Rn.git
   cd calculadora-rn
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Instale o React Native Paper:
   ```bash
   npm install react-native-paper
   ```

## Como Rodar

1. Inicie o servidor Expo:
   ```bash
   npx expo start
   ```

2. Use o aplicativo Expo Go no seu dispositivo móvel para escanear o QR code, ou pressione:
   - `i` para iOS Simulator
   - `a` para Android Emulator

## Estrutura do Projeto

```
calculadora-rn/
├── App.js          # Componente principal da aplicação
├── app.json        # Configurações do Expo
├── index.js        # Ponto de entrada
├── package.json    # Dependências e scripts
├── assets/         # Recursos estáticos
└── README.md       # Este arquivo
```

## Como Usar

1. Abra o aplicativo no seu dispositivo ou emulador
2. Digite os números usando os botões numéricos
3. Selecione a operação desejada (+, -, *, /)
4. Digite o segundo número
5. Pressione "=" para ver o resultado
